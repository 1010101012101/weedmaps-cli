# weedmaps-cli
Find dank shit from the terminal

## Usage
`weedmaps-cli <location> [filters]`

```bash
# All filters specified
weedmaps-cli sacramento --indica --sativa --hybrid --concentrate --drink --clone --gear --topicals --preroll

# Only show indicas per shop
weedmaps-cli san-jose --indica
```
